
# Finances Intelligentes

Site statique hébergé sur **GitHub Pages** : https://bouhya974.github.io/finances-intelligentes/

## Mettre à jour le site
1. Modifiez vos fichiers.
2. macOS/Linux : `./update.sh` — Windows : `update.bat`.

## Google Analytics 4
- Remplacez `G-XXXXXXXXXX` par votre ID GA4.

## Domaine personnalisé
1. CNAME `www → bouhya974.github.io` dans votre DNS.
2. Settings → Pages → **Custom domain** : `www.votredomaine.com`.
3. Ajoutez un fichier **CNAME** à la racine du repo (contenu : votre domaine).
4. Activez **Enforce HTTPS**.

## SEO
- Open Graph, Twitter Cards, canonical, sitemap inclus.
- Soumettez `https://bouhya974.github.io/finances-intelligentes/sitemap.xml` dans Google Search Console.
